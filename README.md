# gralue

## try it using a hosted instance!
[Vercel instance](https://gralue.vercel.app/)

## run it locally! 
1. clone source code
```
git clone https://github.com/Marguesto/gralue.git && cd ./gralue
```
2. install the dependecies
```
npm install 
```
3. run a wepback-server
```
npm run serve
```
